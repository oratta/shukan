# Smitch LP — Codex App セクション別プロンプト集

## 用途

LP 全体 mockup（1 枚絵）では各セクションが潰れる問題を回避するため、リサーチ §2.4 / §3.1 に従って**セクションを個別に高解像度生成**する。
1 セクション = 1 メッセージで Codex App に投げ、8 回分のチャットで全セクションを生成する。

## 使い方

各セクションのメッセージは以下の手順で送信:

1. **Style anchor 画像を添付**: ユーザーが Codex App で作った **LP 全体 mockup 1 枚のみ**を添付（既存の hero.png / problem-solution.png / evidence.png はメッセージ性が薄いので使わない）
2. **共通ヘッダ + 該当セクションのプロンプト**を 1 つのメッセージとしてコピペ
3. **1 セクション 1 枚（1792x1024 landscape）** で出力依頼

全 8 セクション揃ったら、Claude Code に渡す → HTML/CSS で忠実に再現。

## 共通ヘッダ（毎回コピペ、各セクションプロンプトの冒頭に置く）

```
## Style Anchor

添付した LP 全体 mockup を style anchor とします。以下を必ず参照:
- 配色（Deep Indigo #2B4162 + Warm cream #F8F9FA + Warm wood tone）
- タイポグラフィ（editorial serif + humanist sans, NO Inter / Roboto）
- 人物トーン（30-40 代の日本人 / 東アジア系ナレッジワーカー、editorial documentary 風、内省的、自然光、カメラ目線なし、笑顔のステレオタイプなし）
- 構図（editorial asymmetry, 写真素材と iPhone モックを組み合わせるリッチな editorial 仕上げ）

mockup と完全に整合するスタイル・色味・人物の雰囲気・iPhone モックの UI 方向で、以下に指定するセクションだけを高解像度で生成してください。

## Smitch の核（絶対にブレてはいけない）

- "Switch your path." — エビデンスベースで「なりたい自分」を選び取る、人生切替アプリ
- 受動（SNS / ショート動画で流れてきたライフハックを鵜呑み）→ 能動（自分の意思で選ぶ）への転換
- 静かに寄り添う、煽らない、押し付けない、editorial で knowledge-worker 向け
- ストリーク否定、ゲーミフィケーション否定

## 禁止事項（画像に絶対含めない）

- NO 紫グラデーション on 白背景（AI slop の代表）
- NO tech bro neon / cyberpunk
- NO Inter / Roboto / Space Grotesk / Arial / system fonts が可視
- NO 3 カラム rounded-card feature grid（AI slop の典型）
- NO glass orb / 3D blob / smooth gradient without texture
- NO 笑顔の generic stock-photo people / SNS-flex な構図
- NO eye-contact-with-camera（人物が出るなら横顔・後ろ姿・伏し目）
- NO flexing / showing-off body language
- NO streak counters / gamification badges
- NO watermark / borders / studio logos
- NO perfect symmetry
- NO 植物擬人化（「習慣を育てる」系メタファー）

## 写真素材の方針（重要）

「本とコーヒーと眼鏡」のような無人静物に逃げない。**人物写真を主役**にして感情を揺さぶる:
- 都会で生きる 30-40 代の knowledge worker
- 内省的、静かな決意、過去の習慣アプリで疲弊した瞬間、自分で選び取る瞬間
- editorial documentary photography、自然光、shallow depth of field
- 多様性（男女、年代、シーン）を担保するが、トーンは統一

iPhone モックは LP mockup の方向性（Deep Indigo + Cream の Smitch アプリ UI）を踏襲。
```
