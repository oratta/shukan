# 動作確認ガイド

## 環境
- URL: http://localhost:3000 (または https://shukan-nine.vercel.app)
- 起動コマンド: `npm run dev`

## 確認項目（spec Scenarios より）

### review-history

- [ ] Stats ページを開く → 下部に「振り返り履歴」セクションと月間カレンダーが表示される
- [ ] カレンダーの曜日ヘッダーが locale に対応している（ja: 月火水..., en: Mon Tue Wed...）
- [ ] レビュー入力済みの日にムード色ドットが表示される（mood 4-5=緑, 3=黄, 1-2=赤）
- [ ] レビュー未入力の日にはドットなし
- [ ] 前月ボタン（←）をタップ → 前月カレンダーに切り替わる + 選択日がリセットされる
- [ ] 現在月では次月ボタン（→）が無効化されている
- [ ] 未来の日付がテキスト色薄く、タップしても反応しない
- [ ] レビューデータのある日をタップ → カレンダー直下にムード（Lucide アイコン + 色）+ コメント + 習慣ステータスが展開される
- [ ] 同じ日を再タップ → 詳細パネルが閉じる
- [ ] 別の日をタップ → 詳細がその日の内容に切り替わる
- [ ] 習慣のステータスアイコン: ✓(completed/緑) / ✗(failed/橙) / −(skipped/灰) / ○(none/薄灰)
- [ ] 習慣にメモがある場合、習慣名の下にメモが表示される
- [ ] データがない日をタップ → 「この日の記録はありません」メッセージ
- [ ] MOOD_ICONS が yesterday-review-sheet と共有されている（yesterday review シートのムードアイコンが壊れていないこと）

## 受け入れ条件（instruction.md より）

- [ ] ビルドエラーなし
- [ ] 既存の Stats コンテンツが壊れていない（上部のストリーク・達成率・インパクトがそのまま）
