Smitch Logo Concepts

M/W flip + Deep Indigo (#2B4162)

A: Geometric M/W Path

M と W が一筆書きで繋がる。道(path)の暗示。

smitch

上向き = M(ichi)。反転すると下向き = W。一筆書きが「道」を暗示。最もシンプルで汎用性が高い。

B: Soft M/W + Dot

丸みのあるM/W。ドットが「気づき」を表現。

smitch

曲線で柔らかさを出し、中央上のブルードットが「気づき」「目標」を暗示。「静かに寄り添う」トーンに合う。

C: Path Switch

分岐する道。「道を切り替える」を直接表現。

smitch

下から上へ向かう道が分岐点(blue dot)で切り替わる。点線は「選ばなかった道」。Switch の概念を最も直接的に表現。

D: S-Path

Sの文字自体が曲がりくねった道。最もミニマル。

smitch

S字カーブ = 道の曲がり。上のblue dot = 出発点(現在)、下のgreen dot = 到達点(成長)。最もミニマルで小さいサイズでも識別可能。
