- UIの調整
	- タップしたらタップした習慣をハンバーガー的に開く
		- 開いたら表示がこうなる
			- 習慣名(元々のリストカードで表示している)
			- 直近5日間の成果(元々のリストカードで表示している)
			- 習慣がもたらす人生への意義
			- 現在の連続日数(30を100として%付き)
				- 30日を超えたら王冠マークとか、リボンマークとかで装飾した状態で、100%以上の数字としてカウントし続ける
			- ロケットを使える時にロケット使えるボタン(ロケットに関しては詳細ビューで説明)
			- 詳細ビューへのボタン。タップすると詳細ビューが開く
	- 詳細ビュー
		- 広めのモーダルで表示
		- 枠外タップor右上の×ボタン or ブラウザ/アプリの戻る機能で閉じる
		- 習慣名
		- 習慣がもたらす人生への意義
		- 現在の連続日数+%ステータスつき
		- 過去履歴での連続日数の最高記録+％ステータス
		- 10日で一度貯まるロケットのあと何日で貯まるのかと、持っているロケットの数
		- 過去7日を1列として、複数列を使って開始から今までの全ての日数が表示されている
			- この画面ではトグルできない=ホームでしかトグルできないので5日経つともう入力できない
			- 5日たった未入力は×に変わっている
			- ロケットを使える場合、直近5日以降で最初の「やってなかった」日にロケットマークが点滅している
				- タップするとエフェクトが入ってその日が達成したことになる(ロケットマークが小さく入って、ロケットを使ったことを明示)
				- ロケットで達成にした後に、連続日数や%ステータスが更新される
		- 所持しているロケットの数と、ロケットゲットまでの日数
				- 10日連続でロケットを一個ゲット
					- ロケットを使うことで過去の「やらなかった日」を「やったことに修正」することができる。ロケットは詳細ビューで使う
	- 従来のタスクカードの修正
		- 「VS誘惑ボタン」
		- 押すと誘惑に対抗するためのリストのモーダルが出てくる
			- 従来の機能では「やらない系の習慣」の時に「Urge Hit」として出てきたやつをここで表示
			- デイリータップは達成or未達成のトグルにする
		- 対抗リストのモーダルでは
			- 対抗するためのリスト
			- 全部達成すると、「成功エフェクト」が出てモーダルがとじる
			- 全部達成する前にも「成功ボタン」的なものがあってそれを押すと、モーダルがとじる。リストを全部実行する前に誘惑に精神的に打ち勝った時に使う
			- 今日何回誘惑対抗リストを実行したかと、N回誘惑に対抗できたら達成と満たす数を分数として+プログレスバーで表示しておく
		- やらないことリストのデイリーの丸に関して
			- タップすると、やった、やってないのトグルになる
			- 表示上は誘惑への対抗を何回成功したかを分子、設定されている一日の「N回誘惑に対抗できたら達成とみなす」数を分子とした円のプログレスゲージになっている(数字は出さなくてもいい)
			- 定義されたN回(デフォルトは3)をクリアしたら、タップしてなくても「やった扱い」になる
			- 誘惑対抗によりやった扱いになっても、タップでやったやってないをトグルすることはできる。
		- 横幅が狭くなって、習慣名が見えなくなることを防ぐ
			- 習慣名と5日分の入力項目の関係を経て関係にする
			- それでも表示しきれなかったら改行する
			- そもそも、一般的なスマホのサイズで改行して2行で収まる文字数を習慣名の文字数上限とする
